# JPTagView
Customized tag pages

> 使用 CocoaPods

```
pod 'JPTagView' 

或者直接使用 GitHub 链接
pod 'JPTagView', :git => "https://github.com/baiyidjp/JPTagView.git", :tag => '1.6.5'


在使用的页面中 引用 #import "JPTagView.h"

```

> 下载使用

```
直接将下载文件中的 JPTagView 文件夹拖入项目中

在使用的页面中 引用 #import "JPTagView.h"
```
